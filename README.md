### Custom Cape Mod

\-------------------------------------------------

(Attention, This README is partly generated by AI tool as the developer is not English-speaking)

This mod allows users to customize their capes freely.

However, this only affects the local texture on your own screen, and other players in multiplayer games cannot see your custom cape.

This mod was originally made for my own use, but other YouTubers or video-makers can also use it to look better in their videos.
